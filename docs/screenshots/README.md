# Screenshots

Add launch screenshots here before making the repository public.

Recommended files:

- `landing.png`
- `find-partner.png`
- `requests-board.png`
- `session-room-pdf.png`
- `session-room-ai.png`
- `notebook.png`

Guidelines:

- Use realistic but non-private text.
- Do not show real emails.
- Do not show real API keys.
- Do not show private PDFs unless they are public-domain or created for the demo.
- Capture desktop and mobile for the session room.
- Keep the screenshots consistent with the paper-and-ink aesthetic.

Suggested local flow:

1. Start the app with `npm run dev`.
2. Create a test account.
3. Upload or fetch a public-domain PDF.
4. Seed or accept a test pairing.
5. Capture the pages listed above.
